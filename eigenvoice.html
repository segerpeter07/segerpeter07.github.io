Eigenvoice

A voice identification system that uses eigen-based linear algebra and Principal Component Analysis to identify voices.

Final project for Quantitative Engineering Analysis (QEA) at Olin College. Fall 2017.

  • Learn more

What it is

Eigenvoices was a final project for Quantitative Engineering Analysis (QEA) where we explored audio processing and linear algebra in hopes of creating a voice identification algorithm and potentially a voice synthesis algorithm. Building off of a previous QEA unit where we explored linear algebra concepts to create a facial detection algorithm power by principal component analysis of the eigenvectors of an image, we sought to apply this concept to audio processing. Building off another unit where we performed digital signal processing, we wanted to apply the concepts of fourier transforms and frequency domain analysis to this project.

In the end, we were not able to get voice synthesis working, but successfully got the system to identify voices. Using all our datasets, we were able to get an accuracy of ~40%. Read about our results here.

Spectral Subtraction Filtering

Using samples of only background noise along with the whole signal, the frequencies with prevelent noise were modulated to reduce the noise with very accurate results.

  • Learn more

Audio Processing

After collecting a dataset of words, each clip was projected into the frequency domain using a discrete fourier transform before being filtered and trimmed to length.

  • Learn more

Principal Component Analysis

Each word dataset contained three sub-datasets that were used to calculate the covariance matrix. Then the eigenvectors were calculated and the principal components used to identify voices.

Do it yourself

As part of this final project, we created a problem set to help teach the information to other students. This 8 hour problem set covers a review of core linear algebra concepts including:

  • Principal Component Analysis (PCA)
  • Covariance and Correlatio Matrices
  • Single Value Decomposition (SVD)
We also combine these linear algebra concepts with frequency analysis concepts including:
  • Sampling Rates
  • Discrete Fourier Transforms
  • Spectral Subtraction filtering in frequency domain
