FitSpot

A web app that curates a playlist based on your running pace using uploaded accelerometer data from your phone and the Spotify API .

  • Try it out!

Purpose

The purpose of this app is to be able to get a good running or walking playlist without using cellular data. Currently, Spoitfy has a feature that plays songs based on your running pace. However, this requires the use of cellular data, which is not always available or is expensive. With FitSpot, a user can upload accelerometer data from a run and get a Spotify playlist that they can dowload and use on their next run. The intended user-group are runners that do not have access to cellular data regularly when they are running.



Data Collection

Initially we collected data by going on a run and placing a phone in a tight pocket of a fanny pack. The fanny pack was attached to the waist of the runner. We also collected data by attaching a phone to an arm and going on a run to see if a less standard motion could produce the same results.





Data Processing

The accelerometer data from the phone is a csv file that needs to be processed to adjusted due to the presence of gravity, it also needs to be arranged into lists of x, y and z axis values.





Data Analysis

Depending on the orientation of the phone the x or y data was put through a fourier transform. The frequency (besides gravity) with the highest value was taken as the step frequency for steps a second.



Results and Validation

This frequecny was converted to steps per a minute and then divided by two for BPM. We validated that data by running to the generated playlists and determining that they went really well with our running pace.
