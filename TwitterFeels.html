Sentiment Analysis

According to Merriam-Webster Dictionary, 'sentiment' is defined as

a view of or attitude toward a situation or event; an opinion.

This program allows users to choose two twitter users and are given a graph of the resultant sentiment on a scale of -1 to 1. This scale is based off of VADER (Valence Aware Dictionary and sEntiment Reasoner) which is a lexicon and rule-based sentiment analysis tool that is specifically attuned to sentiments expressed in social media, and works well on texts from other domains.

System Architecture

This project was part of a first year software design class taught at Olin College of Engineering in the spring of 2017.

Students were tasked with using an external data source and creating algorithms to extract meaningful results from these large datasets. I decided to use Twitter's API to extract the 300 most recent Tweets and wrote a simple sentiment analysis tool which analyses each Tweet as a whole and calculates a cumulative 'score' for all the tweets. The tool also features a word frequency algorithm that counts the number of words used as well as a ranking of the top 10 most frequently used words.

Data Visualization

Gathering all this information wouldn't have much use if it were not able to be seen in a clear way. I created simple bar graphs that show a representation of the person's net sentiment score. The higher the bar, the more positive their score.
