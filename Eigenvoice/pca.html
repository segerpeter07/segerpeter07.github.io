Pincipal Component Analysis


Principal component analysis (PCA) is a transformation that converts a dataset of possibly correlated variables into a set of values of linearly uncorrelated variables (principal components, or eigenvectors). Picking meaningful principal components is better in comparison because non-essential principal components (eigenvectors with smaller eigenvalues) do not have much information. Since PCA is introduced as a very powerful tool for facial recognition algorithm, we thought it would also be powerful for classifying different voices.

Currently, our PCA algorithm uses a normalized covariance matrix that compares the mean of the voice sample data and the mean of each voice sample data (a person saying a word once). To achieve our goal, we normalized our data by subtracting the generated covariance matrix. Then, we conducted Singular Value Decomposition to sort the eigenvectors in descending order. After sorting, we picked 50 eigenvectors and re-built the data with principal components. Finally, we could find out which dataset has the minimum difference between the mean of the voice data and the mean of each voice sample data.
